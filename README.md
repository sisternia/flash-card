Dẫn tới \Flash Card\Flash-Client nhập những lệnh sau:
- Cài đặt thư viện
  ```bash
  npm i
   ```
- Chạy dự án
  ```bash
  npm run start
   ```

Dẫn tới \Flash Card\Flash-Server
- Cài đặt thư viện
  ```bash
  npm i
   ```
- Chạy dự án
  ```bash
  node app.js
   ```
